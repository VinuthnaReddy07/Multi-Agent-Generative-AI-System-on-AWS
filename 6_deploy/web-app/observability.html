📚 Metrics Guide

🎯 Understanding Your Agent Metrics

This guide explains what each metric means and how to interpret the data in your CloudWatch dashboard.

🚀 Performance Metrics

LatencyMs: Time taken for agent to respond (lower is better)

TotalDuration: Complete processing time including all operations

TotalCycles: Number of reasoning cycles the agent went through

💰 Cost Metrics

InputTokens: Tokens sent to the model (affects cost)

OutputTokens: Tokens generated by the model (affects cost)

ToolCallCount: Number of external tool invocations

✅ Quality Metrics

SuccessRate: Percentage of successful operations (aim for >95%)

ErrorCount: Number of failed operations (should be low)

SuccessCount: Number of successful operations

📈 Best Practices

  • Monitor Latency: Keep response times under 5 seconds for good UX
  • Track Token Usage: Monitor costs by watching input/output token consumption
  • Watch Success Rates: Investigate if success rate drops below 90%
  • Analyze Patterns: Look for trends in usage during different times
